# Driver
A navigator to actuate the turtlebot 3 in all directions.

Tested on:
<br> Ubuntu 18.04
<br> ROS Melodic
<br> Turtlebot3 

# Instructions to run:

```
cd catkin_ws/src
git clone https://github.com/RishiTejaMadduri/Robot_Walker.git 
source devel/setup.bash 
cd .. 
catkin_make 
rosrun driver robot_navigator 
```
